# Instruction Manual for Compiling and Running the Code

## Introduction
This code is designed to perform lexical analysis and evaluation of mathematical expressions. The README provides instructions on how to compile and run the code along with information on the provided input file.

## Prerequisites
Before running the code, make sure you have the following prerequisites installed on your system:
- Python 3.x

## How to Compile
No compilation is required for this Python code. Just click run.

## How to Run
1. Open a terminal or command prompt.
2. Navigate to the directory containing the code files.

### Running the Code
Execute the following command:
python main.py
